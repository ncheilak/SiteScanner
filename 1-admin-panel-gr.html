☁ Ρύθμιση Dropbox Guide

Your worker database is a single workers.json file. Store it in a shared Dropbox folder so every team member uses the same data.

1
Go to Workers tab → click "Save DB to File". This downloads workers.json.
2
Move workers.json into a shared Dropbox folder — e.g. Dropbox/SiteLog/.
3
Share that Dropbox folder with all site engineers who need to run the Scanner App.
4
Each morning, the scanner app lets the engineer Load DB from File — they pick workers.json from Dropbox on their device.
5
At end of day, the completed log can be exported as CSV and saved back to Dropbox for your records.
6
When you add new workers, repeat step 1 to overwrite workers.json in Dropbox.
💡 TIP — DROPBOX MOBILE

On Android/iOS, install the Dropbox app and enable "Make Available Offline" for the SiteLog folder. The scanner can then load the file even without internet at the site.
